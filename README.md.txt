# 🌦️ Weather App

A simple weather application that shows current weather based on user location or city search.

## 🚀 Features

- Shows temperature, wind speed, humidity, and clouds
- Supports geolocation and manual city search
- Responsive and mobile-friendly
- Uses OpenWeatherMap API

## 🧪 Tech Used

- HTML, CSS, JavaScript
- OpenWeatherMap API
- Browser Geolocation API
